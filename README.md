# bumblebeeRobbing
Scripts and data for the manuscript 'Yeast volatiles promote larceny in bumblebee behavior'.

Scripts are run in R and data saved as comma separated values. 
bumblebeeRobbing_script.Rmd is the code used to analyze behavioral and olfactometer data, written by Daniel Souto
Analysis_NectarVials_RobberBehavior_Jan2026.Rmd is the code used to analize volatile data from the GCMS, written by Valerie Martin

Please make sure to adapt the code to your own pertinent working directories.

Any doubts or questions please send them to daniel.souto.v@gmail.com
